Visualization of Problems Solved by Dynamic Programming and Traditional Recursion

Jay Agrawal, Lurein Perera, Eric Wang, and Brian Ward
Algorithms (CSCI3383), 2019

College-level Computer Science students are typically familiar with recursion as a go-to method of completing homework assignments or projects. Unfortunately, there exist more time-efficient techniques such as Dynamic Programming that aren't touched upon until later 3000 level courses. That being said, the aim of this project was to create a beginner-friendly way of comparing the efficiency of Dynamic Programming and recursion simultaneously. The UI presents users with a sample problem like the "Fibonacci Sequence" and runs both Dynamic Programming and recursive calls to the problem. Not only does the program output the results, but it also shows the respective run-times of each method.
